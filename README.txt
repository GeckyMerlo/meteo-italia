🌐 API UTILIZZATE:

1. 🔵 Open-Meteo GFS
   - Modello americano (NOAA)
   - Previsioni ogni ORA (24h/giorno)
   - Include precipitazioni dettagliate

2. 🟢 Open-Meteo ECMWF
   - Modello europeo professionale
   - Più accurato per l'Europa
   - Previsioni ogni ORA

3. 🟣 wttr.in
   - Servizio open source
   - Previsioni ogni 3 ORE
   - Descrizioni in italiano

═══════════════════════════════════════════════════════════════

🕐 PREVISIONI ORARIE:

Clicca su QUALSIASI card meteo per vedere:

• Temperatura ora per ora
• Umidità relativa (%)
• Velocità vento (km/h)
• Probabilità pioggia (%)
• Precipitazioni previste (mm)
• Condizioni meteo con icone

═══════════════════════════════════════════════════════════════

📊 DATI DISPONIBILI:

Previsioni Giornaliere (tutte le API):
├─ Temperatura min/max
├─ Condizioni meteo
├─ Vento
├─ Umidità
└─ Precipitazioni

Previsioni Orarie (click sulla card):
├─ Open-Meteo GFS: 24 ore × 7 giorni
├─ Open-Meteo ECMWF: 24 ore × 7 giorni
└─ wttr.in: 8 slot (ogni 3h) × 3 giorni

❓ FAQ:

Q: Serve una connessione internet?
A: Sì, per scaricare i dati meteo dalle API.

Q: I dati sono reali?
A: Sì! Provengono da modelli meteorologici professionali.

Q: Perché le 3 API mostrano temperature diverse?
A: Usano modelli meteo diversi. ECMWF è generalmente più 
   preciso per l'Europa, GFS meglio per precipitazioni.

Q: Devo installare Node.js?
A: NO! Basta aprire index.html nel browser.

Q: Posso aggiungere altre città?
A: Il sito supporta TUTTE le città italiane automaticamente.

Q: Quanti giorni di previsioni?
A: Fino a 7 giorni in avanti.

Q: Le previsioni orarie funzionano per tutti i giorni?
A: Sì! Anche per i giorni futuri (+7 giorni).

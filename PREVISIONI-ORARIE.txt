📈 DATI ORARI DISPONIBILI:

Ogni ora mostra:

🕐 Ora (formato 24h)
🌡️ Temperatura
💧 Umidità (%)
💨 Velocità vento (km/h)
☔ Probabilità pioggia (%)
🌧️ Precipitazioni previste (mm)
🌤️ Condizioni meteo (icona + descrizione)

═══════════════════════════════════════════════════════════════════════

🔵 OPEN-METEO GFS - Dati Orari:

✓ Previsioni ogni ORA (24 ore al giorno)
✓ Temperature precise ora per ora
✓ Umidità relativa
✓ Probabilità e quantità precipitazioni
✓ Vento: velocità e direzione
✓ Condizioni meteo dettagliate

Esempio:
- 00:00 → 15°C, 70%, 5km/h, ☁️ Nuvoloso
- 01:00 → 14°C, 72%, 6km/h, ☁️ Nuvoloso
- 02:00 → 14°C, 74%, 5km/h, 🌧️ Pioggia leggera
...fino alle 23:00

═══════════════════════════════════════════════════════════════════════

🟢 OPEN-METEO ECMWF - Dati Orari:

✓ Previsioni ogni ORA
✓ Modello europeo più preciso
✓ Temperature ora per ora
✓ Velocità vento
✓ Condizioni meteo

Nota: ECMWF non include umidità e precipitazioni nei dati orari,
      ma è più accurato nelle temperature!

═══════════════════════════════════════════════════════════════════════

🟣 WTTR.IN - Dati Orari:

✓ Previsioni ogni 3 ORE (8 slot al giorno)
✓ Temperature
✓ Umidità
✓ Probabilità pioggia
✓ Precipitazioni
✓ Vento con direzione
✓ Descrizioni dettagliate in italiano

Orari: 00:00, 03:00, 06:00, 09:00, 12:00, 15:00, 18:00, 21:00

❓ FAQ:

Q: Perché wttr.in mostra solo 8 ore invece di 24?
A: wttr.in fornisce dati ogni 3 ore (è comunque molto accurato!)

Q: Quale API è più precisa per le previsioni orarie?
A: ECMWF per le temperature, GFS per precipitazioni dettagliate

Q: Posso stampare le previsioni orarie?
A: Sì! Il browser può stampare il contenuto del modal

Q: Funziona offline?
A: No, richiede connessione internet per caricare i dati

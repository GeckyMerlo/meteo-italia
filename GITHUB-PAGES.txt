🛠️ TROUBLESHOOTING:

PROBLEMA: Il sito non appare dopo 5 minuti
SOLUZIONE: 
  → Controlla Settings → Pages → Source
  → Verifica che il branch sia "Silvio"
  → Controlla Actions tab per errori build

PROBLEMA: 404 Not Found
SOLUZIONE:
  → Assicurati che index.html sia nella root
  → Verifica che .nojekyll esista
  → Fai un nuovo push per forzare rebuild

PROBLEMA: CSS/JS non caricano
SOLUZIONE:
  → Usa path relativi (già fatto ✓)
  → Verifica che i file siano committati
  → Controlla la console browser (F12)
